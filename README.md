# Very Parivarik Show Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard developed to analyze data related to the "Very Parivarik" show. The dashboard provides meaningful insights through visualizations, helping users understand trends, engagement, and performance metrics.

## Objectives

- Analyze show-related data effectively.
- Visualize audience and content trends.
- Generate actionable insights through interactive reports.
- Demonstrate Power BI dashboard development skills.

## Features

- Interactive Dashboard
- Episode-wise Analysis
- Trend Visualization
- Performance Metrics
- Dynamic Filters and Slicers
- Data-Driven Insights

## Tools & Technologies

- Power BI
- DAX
- Data Modeling
- Data Visualization
- Business Intelligence
